How to Track Missing Subjects

1

Register a Target

Navigate to the Dashboard and use the registration form. Provide the subject's legal name and upload a clear, front-facing photograph without sunglasses or masks. The system instantly computes an encrypted biometric embedding of their face.

Pro Tip: Images sourced from ID cards yield >98% true-positive matches across varying lighting conditions.
2

Configure Data Streams

Navigate to the Live Detection tab. Deploy streams dynamically by adding RTSP endpoints linked to your CCTV system or drone payloads. Enter the IP configuration in the Add Stream panel.

3

Monitor & Alert

Launch the stream by pressing Start. The dashboard binds WebSocket connections to the inference pipeline. YOLO face detection runs locally with InsightFace ArcFace recognition.

Matches exceeding a 0.55 similarity threshold trigger instantaneous alerts — both visual toasts and audible notifications.

Audit Compliance: All detections and alerts are logged chronologically and preserved in the Admin Panel for law enforcement review.
4

Manual Accuracy Testing

Use the Camera Test page for offline verification. Field agents can capture snapshots from laptop webcams and test them against the master embedding database. This is useful for validating model accuracy and testing detection quality.
